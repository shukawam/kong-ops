# 📋 利用規約とポリシー

Toshogu APIを安全かつ適切に利用するための規約とポリシーをご確認ください。

::alert
---
type: "warning"
---
**重要:** APIを使用することで、これらの規約に同意したものとみなされます。
::

---

## 📜 利用規約

### 1. アカウントとアクセス

#### 1.1 アカウント登録

- 正確かつ完全な情報を提供すること
- アカウント情報を最新の状態に保つこと
- 1人につき1つのアカウントのみ作成可能
- 18歳以上であること

#### 1.2 API Keyの管理

**✅ 許可される使用**

- セキュアな環境でのAPI Keyの保管
- 環境変数や秘密管理システムの使用
- 定期的なKeyのローテーション
- チーム内での適切な共有

**❌ 禁止される使用**

- API Keyの公開リポジトリへのコミット
- クライアントサイドコードへの埋め込み
- 第三者への無断譲渡
- 複数アカウントでの共有

#### 1.3 アカウントのセキュリティ

開発者の責任：

- 強力なパスワードの使用
- 二要素認証の有効化（推奨）
- 不正アクセスの速やかな報告
- アカウント情報の機密保持

---

### 2. 利用制限

#### 2.1 レート制限

各プランのレート制限を遵守すること：

| プラン | 時間あたり | バースト |
|--------|-----------|---------|
| Free | 1,000 | 50 |
| Pro | 10,000 | 200 |
| Enterprise | カスタム | カスタム |

> 💡 **注意:** レート制限を意図的に回避する行為は禁止されています。

#### 2.2 公正な利用

以下の行為は禁止されています：

- **過度なリクエスト**: 必要以上に頻繁なAPI呼び出し
- **リソースの独占**: 他のユーザーに影響を与える使用
- **スクレイピング**: APIの目的外利用
- **DDoS攻撃**: 意図的なサービス妨害

#### 2.3 データ利用制限

- **キャッシング**: 取得したデータは24時間までキャッシュ可能
- **再配布**: 取得したデータの第三者への販売・譲渡は禁止
- **バックアップ**: 個人利用目的のバックアップは許可

## 🔒 プライバシーポリシー

### データ収集

当社が収集する情報：

#### 3.1 アカウント情報

- 氏名、メールアドレス、会社名
- ログイン情報（パスワードはハッシュ化）
- 支払い情報（決済プロバイダー経由）

#### 3.2 利用データ

- APIリクエストログ（IPアドレス、タイムスタンプ、エンドポイント）
- エラーログとデバッグ情報
- パフォーマンスメトリクス

#### 3.3 Cookieとトラッキング

- セッション管理用Cookie
- アナリティクス（Google Analytics使用）
- 広告目的でのトラッキングは行いません

### データの使用目的

収集したデータは以下の目的で使用：

- **サービス提供**: API機能の提供と維持
- **サポート**: 技術サポートとトラブルシューティング
- **改善**: サービス品質の向上
- **セキュリティ**: 不正利用の検知と防止
- **コンプライアンス**: 法的要件への対応

### データの保護

| 項目 | 対策 |
|------|------|
| 🔐 **暗号化** | 転送時: TLS 1.3 / 保管時: AES-256 |
| 🏢 **アクセス制御** | 最小権限の原則 / 多要素認証 |
| 🔍 **監査** | 定期的なセキュリティ監査 / 侵入テスト |

### データの保持期間

| データ種別 | 保持期間 |
|-----------|---------|
| アカウント情報 | アカウント削除まで |
| APIリクエストログ | 90日間 |
| エラーログ | 30日間 |
| 支払い情報 | 7年間（法的要件） |

### ユーザーの権利

GDPR、CCPAに基づく権利：

- **アクセス権**: 保存データの開示請求
- **修正権**: 不正確なデータの修正
- **削除権**: データ削除の要求（Right to be forgotten）
- **移行権**: データのポータビリティ
- **異議申立権**: データ処理への異議

**権利行使の方法:** [privacy@toshogu.example.com](mailto:privacy@toshogu.example.com)

## 🌍 コンプライアンス

### 5.1 地域規制

#### GDPR（EU一般データ保護規則）

- データ処理の合法性確保
- データ主体の権利保護
- データ侵害時の72時間以内通知

#### CCPA（カリフォルニア州消費者プライバシー法）

- 消費者のプライバシー権保護
- データ販売のオプトアウト権
- 差別の禁止

#### 個人情報保護法（日本）

- 個人情報の適切な取り扱い
- 安全管理措置の実施
- 本人同意の取得

### 5.2 業界標準

準拠している標準：

- **ISO 27001**: 情報セキュリティ管理
- **SOC 2 Type II**: サービス組織統制
- **PCI DSS**: 決済カード業界データセキュリティ基準

### 5.3 セキュリティ対応

#### 脆弱性報告

セキュリティ脆弱性を発見した場合：

1. **報告先**: [security@toshogu.example.com](mailto:security@toshogu.example.com)
2. **情報提供**: 再現手順、影響範囲、概念実証（可能な場合）
3. **対応**: 24時間以内に初期応答、30日以内に修正

> ⚠️ **責任ある開示:** 脆弱性の公開は修正後にお願いします。

#### インシデント通知

データ侵害が発生した場合：

- 影響を受けるユーザーへの72時間以内通知
- 侵害内容と影響範囲の説明
- 取られた対策の報告
- 今後の防止策の提示

## 🚫 禁止事項

以下の行為は明示的に禁止されています：

### 技術的禁止事項

- **リバースエンジニアリング**: APIの逆アセンブル、逆コンパイル
- **セキュリティ回避**: 認証メカニズムの回避
- **負荷テスト**: 事前承認なしの負荷テスト
- **自動化**: レート制限を超える自動化スクリプト

### 利用目的の禁止事項

- **違法行為**: 法律に違反する用途
- **スパム**: 迷惑メール、スパムの送信
- **マルウェア**: 悪意のあるソフトウェアの配布
- **詐欺**: 詐欺的な活動
- **差別**: 差別的なコンテンツの作成

### データ利用の禁止事項

- **無断収集**: APIの目的外でのデータ収集
- **転売**: 取得データの商業的転売
- **プロファイリング**: 個人の詳細なプロファイル作成
- **トラッキング**: ユーザー行動の追跡

## ⚠️ 違反時の対応

### 警告プロセス

1. **第1段階**: 警告メールの送信
2. **第2段階**: アカウント一時停止（7日間）
3. **第3段階**: アカウント永久停止

### 即時停止対象

以下の場合は警告なしに停止：

- 重大なセキュリティ違反
- 違法行為
- DDoS攻撃
- 複数回の違反

### 異議申し立て

アカウント停止に異議がある場合：

1. [appeals@toshogu.example.com](mailto:appeals@toshogu.example.com)へ連絡
2. 事情説明書の提出
3. 14日以内に審査結果の通知
